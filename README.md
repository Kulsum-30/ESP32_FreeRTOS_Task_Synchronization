# ESP32-S3 FreeRTOS Task Synchronization and CPU Utilization Analysis

## Overview

This project analyzes task synchronization and CPU utilization using FreeRTOS on ESP32-S3.

### Hardware Used
- ESP32-S3
- DHT11 Sensor
- MAX30100 Sensor

### Concepts Used
- FreeRTOS Tasks
- Task Priorities
- Binary Semaphore
- CPU Utilization Analysis
- Single Core vs Dual Core Execution
- I2C Communication
